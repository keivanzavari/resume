love what I do

I received a KUKA robot, in two months I learned how to program the robot, how to communicate with it and finally developed a piece of real-time software to fully control the robot using an external PC.

  • The external PC can be running ROS or Orocos
  • The communication is based on UDP
  • KUKA calls this RSI (robot sensor interface)
  • The communication details are defined on the robot using some XML files
  • I hope I can open source the code later
